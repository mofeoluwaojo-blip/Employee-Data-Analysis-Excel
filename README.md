# Employee Data Analysis Dashboard

## Overview
This project analyses employee data using Microsoft Excel.

## Features

- PivotTables
- PivotCharts
- Interactive Dashboard
- Employee analysis by country
- Employee analysis by city
- Employee analysis by company

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
